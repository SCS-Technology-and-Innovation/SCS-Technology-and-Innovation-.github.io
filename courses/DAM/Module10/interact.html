Typing style matters

ⓘ

This module is synchronous during class session, but recorded for asynchronous viewing.

We have a handwriting even on a keyboard

Type the same text several times, at least once per each method, while in the first stage "Providing sample data". If you make a big mistakes, you can click on "Erase samples" to start over.

Then, switch the stage to "Comparing to sample data". Write in a single method and then press "Analyze" to see the result. To try another method, use "Reset analysis" and then type in a second method, reanalyze, etc.

If you want to clear the text box, you can either select and delete or click on the "Clear text" button.
