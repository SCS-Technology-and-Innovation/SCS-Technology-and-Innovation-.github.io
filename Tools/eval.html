Course evaluations

In order to use this tool with real data, you need to extract the data from the PDFs downloaded from Mercury into a CSV file with this script. No data is ever uploaded anywhere, all processing is done locally. The data shown when this page loads is artificially generated so you can explore the visualization before extracting and using the real data for your unit.

Term:

Each axis of the spiderweb is the scale of the corresponding question in the course evaluation.

  1. Strongly disagree
  2. Disagree
  3. Neutral
  4. Agree
  5. Strongly agree

The color code of the filling of the spiderweb is as follows:

 Minimum (0 %)

 First quartile (25 %)

 Median (50 %)

 Third quartile (75 %)

 Maximum (100 %)
