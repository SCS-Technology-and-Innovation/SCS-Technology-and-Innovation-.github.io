Course evaluations

In order to use this tool with real data, you need to extract the data from the PDFs downloaded from Mercury into a CSV file with this script (ask for help to set up the environment on your computer for the tool to run properly). No data is ever uploaded anywhere, all processing is done locally on your own computer.

Each axis of the spiderweb visualization corressponds to a question in the course evaluation. The scale of all questions is:

  1. Strongly disagree
  2. Disagree
  3. Neutral
  4. Agree
  5. Strongly agree

The color code of the filling of the spiderweb is as follows:

 Minimum (0 %)  First quartile (25 %)  Median (50 %)  Third quartile (75 %)  Maximum (100 %)
