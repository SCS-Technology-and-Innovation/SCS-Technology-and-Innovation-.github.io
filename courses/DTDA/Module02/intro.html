Introduction

In the previous module, we gained some fluency in reading source code and making minor modifications to it. Our visible output was mostly white lines on a black background. We will now revisit what we learned, instead placing colorful pixels in a rectangular canvas. Our context for employing these new skills is territorial design: we will calculate Voronoi cells as models for delivery zones for pizza restaurants.

Objectives

In this module, you will have an opportunity to...

  • ... draw on a canvas
  • ... calculate distances
  • ... produce a return value from a subroutine
  • ... group values in tuples
  • ... express colors in RGB
  • ... generate pseudorandom numbers
  • ... employ logic to establish conditions
