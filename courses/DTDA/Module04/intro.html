Introduction

In the previous module, we were placing circular antennas in our flat, rectangular town. We now continue focusing on this task, adding richness to our model. Any antennas with overlapping ranges are likely to interfere with each other's signals. We will need to assign each pair of overlapping antennas different channels (bands of wavelengths) to communicate on so that they can both operate simultaneously without issue.

Learning outcomes

This module will help you do the following:

  • Build upon existing code to create a richer model
  • Model a complex system in terms of a graph
  • Employ a sophisticated data structure (in this case, a graph)
  • Apply basic graph-theoretical concepts (vertex, edge, neighbor, degree)
  • Visualize graphs with a network library
  • Apply a combinatorial problem (graph coloring) to a real-world setting
  • Draft a heuristic for a situation that lacks an efficient algorithm
