Introduction

This module is, for many, the first glimpse at source code. An important thing to keep in mind at all times is that our primary goal is to first become fluent at reading code and then gain the ability to modify existing code. You are not required to write any original code from scratch in order to succeed in the course.

Learning outcomes

This module will help you do the following:

  • Play with elemental trigonometry and geometry
  • Make use of libraries
  • Call existing subroutines
  • Store values in variables
  • Iterate in a loop
  • Store values in a list
  • Define an original subroutine
  • Carry out a recursive calculation

Useful knowledge to freshen up on before class would be the concept of angles (in degrees, 0 to 360 — for example, a right angle is a 90-degree angle). Having a pen and paper (or a tablet) at hand for drawing during class may also prove useful.

Warm-up

Warm-up assessment

Based on the video, describe in your own words what makes a structure into a fractal and name at least one example that comes to mind, outside of what is mentioned in the video, that you believe to have a fractal structure, elaborating in quick strokes on what about it makes you feel like it is fractal?

Concepts

After this module, you should be familiar with the following concepts:

  • Instruction (command)
  • Variable
  • Loop
  • Iteration
  • Recursion
  • Subroutine (function)
  • Fractal

Remember that you can always look concepts up in the glossary. Should anything be missing or insufficient, please report it.
